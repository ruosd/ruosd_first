==================================================
          产品常见问题解答 (FAQ)
==================================================

【Q1: 如何申请发票】
A:
1. 提交订单时勾选"需要发票"
2. 选择发票类型（普通发票/增值税专用发票）
3. 填写发票抬头和税号
4. 发票将在订单签收后3个工作日内开具
5. 电子发票发送至注册邮箱，纸质发票随包裹寄出
6. 如忘记勾选，签收后30天内可在"我的订单"中补开发票

【Q2: 商品尺码如何选择】
A:
1. 查看商品详情页的尺码表（包含胸围、衣长、袖长等数据）
2. 参考"买家秀"中同身材用户的试穿效果
3. 使用"尺码推荐"功能，输入身高体重即可获得建议
4. 如仍不确定，可联系在线客服获取人工建议
5. 支持7天无理由退换，尺码不合适可免费换货

【Q3: 商品有质量问题怎么办】
A:
1. 拍照留存证据
2. 在15天内申请退换货
3. 选择"质量问题"原因
4. 提供照片和问题描述
5. 客服审核通过后，系统自动安排上门取件
6. 确认收货后3个工作日内退款到原支付账户
7. 质量问题产生的运费由商家承担

【Q4: 如何修改订单信息】
A:
- 已支付但未发货的订单可以修改收货信息
- 已发货的订单无法修改，请联系客服协助

【Q5: 如何取消订单】
A:
1. 进入"我的订单"
2. 找到需要取消的订单
3. 点击"取消订单"按钮
4. 选择取消原因
5. 未支付订单立即取消
6. 已支付未发货订单，款项1-3个工作日退回
7. 已发货订单无法直接取消，需先拒收再申请退款

【Q6: 如何查询物流信息】
A:
1. 进入"我的订单"
2. 找到对应订单，点击"查看物流"
3. 显示完整物流轨迹和预计送达时间
4. 如超过预计时间3天未更新，可联系客服查询
5. 支持顺丰、圆通、中通、韵达、京东物流等主流快递

【Q7: 是否支持货到付款】
A:
部分地区支持货到付款，以下单时页面显示为准。货到付款订单需额外支付5元服务费。
