==================================================
          退换货与售后政策
==================================================

一、退换货总则

本店承诺自签收之日起7天内，商品未经使用、包装完整、配件齐全，可申请无理由退换货。15天内出现质量问题可申请换货或维修。以下情况不支持退换货：
- 超过规定期限
- 商品已使用或影响二次销售
- 包装、配件、赠品丢失或损坏
- 食品、内衣、定制类等特殊商品拆封后
- 因个人原因造成的商品损坏

二、退货流程

1. 进入"我的订单"，选择需退货的商品
2. 选择退货原因并上传凭证（照片/视频）
3. 等待客服审核（1个工作日内）
4. 审核通过后，系统安排快递上门取件
5. 商家收到退货后1-3个工作日退款

三、换货流程

1. 进入"我的订单"，选择需换货的商品
2. 选择换货原因和期望换货的规格
3. 等待客服审核
4. 审核通过后，可选择"先寄回再发新"或"先发新再寄回"
5. "先发新再寄回"需支付押金，收到退回商品后退还

四、退款时效

| 支付方式   | 退款到账时间          |
|-----------|---------------------|
| 微信支付   | 1-3个工作日          |
| 支付宝     | 1-3个工作日          |
| 银行卡     | 3-5个工作日          |
| 货到付款   | 5-7个工作日（需提供银行卡信息）|

五、运费承担规则

- 质量问题、发错货：商家承担全部运费
- 7天无理由退换：买家承担寄回运费（商家赠送运费险可抵扣）
- 换货：首次换货运费由商家承担
- 拒收退货：买家承担往返运费

六、售后服务联系方式

- 在线客服：工作日 9:00-21:00，周末 10:00-18:00
- 客服热线：400-XXX-XXXX
- 投诉邮箱：complaints@example.com
- 投诉处理时效：普通投诉5个工作日，紧急投诉2个工作日

七、常见问题

【Q1: 退货后多久能收到退款】
A: 商家签收退货后1-3个工作日完成退款审核，退款按上述时效到账。

【Q2: 收到的商品与图片不符怎么办】
A: 请拍照联系客服，核实后按"描述不符"处理，商家承担来回运费并安排退换。

【Q3: 投诉后多久能得到处理】
A:
- 普通投诉：5个工作日内处理
- 紧急投诉：2个工作日内处理
- 如对处理结果不满意，可升级至平台客服介入
