                     电商客服助手系统使用指南
================================================================================

版本：V1.0
更新日期：2026年6月
适用对象：客服人员、管理员

================================================================================
一、系统概述
================================================================================

电商客服助手系统是一款基于人工智能的智能客服解决方案，旨在帮助电商企业
提升客户服务质量和效率。系统支持多轮对话、订单处理、售后服务等功能。

【核心功能】
1. 智能问答 - 基于知识库的自动问答
2. 订单管理 - 订单查询、状态更新
3. 售后服务 - 退换货处理、投诉管理
4. 数据分析 - 客服数据分析报表

================================================================================
二、系统登录
================================================================================

【普通用户登录】
1. 访问系统首页：http://localhost:3000
2. 点击"用户登录"按钮
3. 输入用户名和密码
4. 点击"登录"按钮进入聊天界面

【管理员登录】
1. 访问系统首页：http://localhost:3000
2. 点击"管理员登录"按钮
3. 输入管理员账号和密码（默认：admin / admin123）
4. 进入管理控制台

================================================================================
三、普通用户功能
================================================================================

【发起咨询】
1. 登录后进入聊天界面
2. 在输入框中输入问题
3. 点击发送按钮或按回车键
4. 系统会自动回复相关答案

【常见问题类型】
- 商品咨询：产品信息、规格参数、价格优惠
- 订单查询：订单状态、物流信息、收货地址
- 售后服务：退换货、维修、投诉建议
- 支付问题：支付方式、退款进度

================================================================================
四、管理员功能
================================================================================

【管理控制台】
管理员登录后可以访问控制台，提供以下功能：

1. 知识库管理
   - 添加新知识文档
   - 更新现有知识内容
   - 删除过时文档
   - 搜索知识条目

2. 客服数据统计
   - 查看咨询量统计
   - 分析热点问题
   - 评估客服质量

3. 用户管理
   - 查看用户列表
   - 管理用户权限
   - 查看用户反馈

4. 系统设置
   - 配置AI参数
   - 设置自动回复规则
   - 管理关键词匹配

================================================================================
五、快捷命令
================================================================================

用户可以使用以下快捷命令获取帮助：

【订单相关】
- "查订单" - 查询最近订单
- "物流" - 查看物流信息
- "收货地址" - 修改收货地址

【商品相关】
- "商品列表" - 查看在售商品
- "商品详情 XXX" - 查询特定商品
- "价格" - 查询商品价格

【售后相关】
- "退换货" - 申请退换货
- "投诉" - 提交投诉
- "建议" - 提交建议

================================================================================
六、常见问题解答
================================================================================

Q1: 忘记密码怎么办？
A: 请联系管理员重置密码。

Q2: 无法登录系统？
A: 请检查网络连接，或联系管理员确认账号状态。

Q3: AI回复不准确？
A: 管理员可以在知识库中更新或添加相关内容。

Q4: 如何导出对话记录？
A: 管理员可以在控制台中导出相关数据。
