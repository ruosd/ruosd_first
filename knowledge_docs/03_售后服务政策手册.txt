二、退换货政策
================================================================================

【七天无理由退换货】
- 适用范围：除特殊商品外，所有商品均支持七天无理由退换货
- 申请时限：签收商品后7天内（以物流签收时间为准）
- 条件要求：
  · 商品及包装完好
  · 附件、赠品齐全
  · 未拆封使用（功能性商品除外）
  · 不影响二次销售

【有理由退换货】
1. 商品质量问题
   - 表现：破损、功能故障、外观瑕疵
   - 处理：免费退换货或维修
   - 时限：签收后15天内

2. 商品与描述不符
   - 表现：颜色、规格、型号等与页面描述不符
   - 处理：退换货，运费由商家承担
   - 时限：签收后7天内

3. 物流损坏
   - 表现：运输过程中造成的商品损坏
   - 处理：拒收或签收后48小时内申请
   - 证明：需要提供照片证据

【特殊商品退换货规则】

1. 服装鞋类
   - 支持七天无理由退换货
   - 退货时需保持商品完好、未经洗涤
   - 内衣裤、泳衣等贴身商品不支持七天无理由退货

2. 数码电子产品
   - 质量问题7天内可退换货
   - 因个人原因（非质量问题）不支持无理由退换货
   - 已激活使用的手机、电脑等不支持七天无理由退货

3. 食品饮料
   - 一般不支持七天无理由退货
   - 质量问题可申请退换货
   - 保鲜期不足的商品可拒收

4. 化妆品护肤品
   - 已拆封使用的不支持七天无理由退货
   - 未拆封且不影响二次销售的可申请退货
   · 使用后过敏需提供医院证明

5. 家具家电
   - 大件商品需预约上门取件
   - 因尺寸问题（非质量问题）退换货需买家承担运费
   - 安装后不支持七天无理由退货

三、退款规则
================================================================================

【退款方式】
1. 原路退回
   - 适用于：微信支付、支付宝、银行卡支付
   - 时效：3-7个工作日

2. 账户余额
   - 适用于：所有支付方式
   - 时效：1-3个工作日

3. 退至银行卡
   - 适用于：银行卡支付（需提供卡号）
   - 时效：5-10个工作日

【退款时效】
- 未发货订单退款：1-3个工作日
- 已发货未签收：3-5个工作日
- 已签收商品退款：收到商品后3-7个工作日
- 节假日顺延

【退款金额计算】
基本退款 = 订单实付金额

退款扣除项：
- 优惠券/红包：不返还
- 赠送积分：从账户扣除
- 运费：一般不退还（质量问题除外）
- 包装费：根据商品情况而定

四、维修政策
================================================================================

【三包服务】
1. 包修
   - 期限：自签收之日起1年内（因品牌而异）
   - 范围：非人为因素造成的功能故障
   - 方式：免费维修或更换配件

2. 包换
   - 期限：自签收之日起15天内
   - 范围：重大质量问题（功能性故障）
   - 要求：需提供检测报告或厂家认定

3. 包退
   - 期限：自签收之日起7天内
   - 范围：严重质量问题影响正常使用
   - 流程：退货→退款

【不属于保修范围】
- 人为损坏（摔坏、进水、私自拆修等）
- 正常磨损和老化
- 不可抗力因素（自然灾害等）
- 超过保修期限的商品

【维修流程】
1. 提交维修申请
2. 客服审核
3. 寄送商品（或上门取件）
4. 厂家检测
5. 维修/更换
6. 返回商品
7. 确认收货

五、投诉处理
================================================================================

【投诉类型】
1. 商品质量问题投诉
2. 服务态度投诉
3. 物流配送投诉
4. 虚假宣传投诉
5. 其他问题投诉

【投诉渠道】
- 在线客服投诉
- 客服热线：400-123-4567
- 邮箱投诉：complaint@example.com
- 微信公众号投诉

【投诉处理流程】
1. 投诉受理（1个工作日内）
   - 记录投诉内容
   - 分配处理人员
   - 反馈投诉编号

2. 调查核实（3个工作日内）
   - 联系相关方
   - 收集证据材料
   - 分析问题原因

3. 处理方案（5个工作日内）
   - 制定解决方案
   - 与投诉人沟通
   - 达成一致意见

4. 执行反馈（7个工作日内）
   - 执行解决方案
   - 跟踪处理结果
   - 反馈给投诉人

5. 归档总结
   - 记录处理过程
   - 分析问题根源
   - 优化改进措施

【投诉处理原则】
- 公平公正，不偏袒任何一方
- 快速响应，及时沟通
- 依法依规，合情合理
- 保护消费者合法权益

六、运费险说明
================================================================================

【运费险概述】
运费险是针对退货场景的保险服务，帮助买家降低退货成本。

【保障范围】
- 七天无理由退货
- 商品与描述不符退货
- 质量问题退货

【不保障范围】
- 换货（只保障退货）
- 二手商品
- 定制品

【理赔流程】
1. 买家发起退货申请
2. 选择"需要运费险"
3. 退货成功后
4. 系统自动理赔
5. 赔付款打入买家账户

【理赔金额】
- 最高赔付限额：25元
- 理赔金额 = 实际退货邮费（以快递公司开具的发票为准）

七、特殊商品售后说明
================================================================================

【定制商品】
- 不支持七天无理由退换货
- 质量问题可申请退换货
- 因买家原因（如尺寸错误）不支持退换货

【进口商品】
- 需要提供购买凭证
- 保税区商品按国家相关政策执行
- 海外直邮商品退换货流程较长

【虚拟商品】
- 一般不支持退换货
- 未使用的充值卡可申请退款
- 电子券、兑换码等不支持退款

【鲜活易腐商品】
- 不支持七天无理由退货
- 签收时发现损坏可拒收或申请售后
- 需要在签收后24小时内申请
