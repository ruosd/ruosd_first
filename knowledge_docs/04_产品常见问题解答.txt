一、账号相关问题
================================================================================

【Q1: 如何注册账号？】
A: 访问系统首页，点击"用户登录"→"注册账号"，填写用户名、邮箱、密码即可完成注册。

【Q2: 忘记密码怎么办？】
A: 在登录页面点击"忘记密码"，输入注册邮箱，系统会发送重置链接到您的邮箱。

【Q3: 如何修改个人信息？】
A: 登录后进入个人中心，可以修改：
- 用户名
- 邮箱
- 收货地址
- 联系电话

【Q4: 账号被锁定怎么办？】
A: 连续输错密码5次后账号会被锁定，30分钟后自动解锁。也可联系客服人工解锁。

【Q5: 如何注销账号？】
A: 联系客服申请注销，需要验证身份信息。注销后所有数据将被永久删除。

二、购物相关问题
================================================================================

【Q1: 如何下单购买？】
A: 
1. 浏览商品，选择心仪的商品
2. 点击"加入购物车"或"立即购买"
3. 确认商品信息和收货地址
4. 选择支付方式并完成支付
5. 等待商家发货

【Q2: 可以使用哪些支付方式？】
A: 我们支持以下支付方式：
- 微信支付
- 支付宝
- 银行卡支付（支持借记卡和信用卡）
- 账户余额支付

【Q3: 支付失败怎么办？】
A: 
1. 检查网络连接是否正常
2. 确认银行卡余额或信用额度充足
3. 确认支付密码或验证码输入正确
4. 稍后重试或更换支付方式
5. 如仍无法解决，联系客服协助

【Q4: 订单可以修改吗？】
A: 
- 未支付订单可以取消或修改
- 已支付但未发货的订单可以修改收货信息
- 已发货的订单无法修改，请联系客服协助

【Q5: 如何取消订单？】
A: 
1. 进入"我的订单"页面
2. 找到需要取消的订单
3. 点击"取消订单"按钮
4. 选择取消原因并确认
5. 已支付金额将在3-7个工作日内退回

三、物流配送问题
================================================================================

【Q1: 如何查询物流信息？】
A: 
1. 进入"我的订单"
2. 点击已发货的订单
3. 查看"物流信息"栏
4. 点击物流单号可查看详细轨迹

【Q2: 一般几天能收到货？】
A: 
- 本地发货：1-3天
- 外地发货：3-7天
- 偏远地区：7-15天
- 具体以页面标注的发货时间为准

【Q3: 可以指定收货时间吗？】
A: 
- 我们无法承诺精确的收货时间
- 可以在备注中填写大致时间要求
- 建议保持手机畅通，以便快递员联系

【Q4: 快递延误怎么办？】
A: 
1. 首先查询物流信息确认状态
2. 拨打快递公司客服电话查询
3. 如遇节假日或恶劣天气，配送可能延迟
4. 超过15天未送达可申请退款或催促

【Q5: 可以自提商品吗？】
A: 部分商品支持自提，您可以在下单时选择：
- 自提点取货
- 门店自提
请注意自提时限，超时将被退回。

【Q6: 包装破损怎么办？】
A: 
1. 签收前检查外包装是否完好
2. 如有破损请拒收并联系客服
3. 已签收请在24小时内拍照并联系客服
4. 提供外包装和商品损坏照片

四、退换货问题
================================================================================

【Q1: 如何申请退换货？】
A: 
1. 进入"我的订单"
2. 点击已完成或已签收的订单
3. 点击"申请售后"按钮
4. 选择退换货类型并填写原因
5. 提交申请并等待审核

【Q2: 七天无理由退货包运费吗？】
A: 
- 购买时有运费险的，可以报销退货运费
- 没有运费险的，自行承担退货运费
- 质量问题退换货由商家承担运费

【Q3: 退换货需要多久处理？】
A: 
1. 提交申请后：1个工作日内审核
2. 审核通过后：3天内寄回商品
3. 收到商品后：3-5个工作日退款/换货

【Q4: 退款多久能到账？】
A: 
- 微信/支付宝：1-3个工作日
- 银行卡：3-7个工作日
- 账户余额：即时到账
- 节假日顺延

【Q5: 商品已过期还能退吗？】
A: 
- 生鲜、食品等有保质期商品：签收后超过24小时不支持
- 服装、数码等商品：支持七天无理由退货
- 具体以商品页面标注的退换货政策为准

五、商品问题
================================================================================

【Q1: 如何了解商品详情？】
A: 
- 商品详情页有详细的商品描述
- 规格参数表展示具体信息
- 购买评价可参考其他买家的反馈
- 可咨询在线客服获取更多信息

【Q2: 收到的商品与描述不符怎么办？】
A: 
1. 保留商品和包装
2. 拍照留存证据
3. 在7天内申请退换货
4. 选择"商品与描述不符"原因
5. 商家承担来回运费

【Q3: 商品有质量问题怎么办？】
A: 
1. 拍照留存证据
2. 在15天内申请退换货
3. 选择"质量问题"原因
4. 提供清晰的损坏照片
5. 商家承担所有运费

【Q4: 如何了解真实库存情况？】
A: 
- 商品页面显示"有货"/"缺货"状态
- 显示具体库存数量（如"库存：50件"）
- 缺货商品到货后会通知您
- 可订阅到货提醒

【Q5: 如何选择合适的尺码/规格？】
A: 
- 查看商品详情页的尺码表
- 参考购买评价中买家的身高体重
- 咨询在线客服获取建议
- 实体店试穿后在线购买

六、支付与发票问题
================================================================================

【Q1: 如何申请发票？】
A: 
1. 提交订单时勾选"需要发票"
2. 选择发票类型（普通发票/增值税专用发票）
3. 填写发票抬头和税号
4. 确认收货后7天内开具并邮寄

【Q2: 已完成的订单可以补开发票吗？】
A: 
- 交易完成90天内可以申请补开发票
- 联系客服提交申请
- 提供订单编号和开票信息
- 发票将在7个工作日内寄出

【Q3: 发票内容可以修改吗？】
A: 
- 普票内容：商品明细或办公用品（不可开现金）
- 增票内容：按实际交易内容开具
- 开具后不可修改，只能作废重开

【Q4: 优惠券如何使用？】
A: 
1. 结算时选择可用优惠券
2. 满足优惠券使用条件（满减门槛）
3. 点击"使用"按钮
4. 系统自动抵扣相应金额
5. 订单完成后优惠券不可退回

【Q5: 积分有什么用途？】
A: 
- 积分可抵扣现金（100积分=1元）
- 部分商品可使用积分兑换
- 积分不可提现
- 积分有效期为获取后12个月

七、售后服务问题
================================================================================

【Q1: 客服的工作时间是什么？】
A: 
- 在线客服：7×24小时
- 电话客服：周一至周五 9:00-18:00
- 节假日服务时间可能调整，请关注公告

【Q2: 如何联系客服？】
A: 
- 在线聊天：点击页面右下角客服图标
- 客服热线：400-123-4567
- 邮箱：service@example.com
- 微信公众号：在公众号内联系客服

【Q3: 投诉后多久能得到处理？】
A: 
- 普通投诉：5个工作日内处理
- 紧急投诉：2个工作日内处理
- 复杂投诉：7个工作日内处理
- 处理结果会通过短信或站内信通知您

【Q4: 如何查看处理进度？】
A: 
1. 进入"我的客服"页面
2. 查看"我的投诉/建议"列表
3. 点击具体投诉单查看处理进度
4. 也可通过订单详情页查看售后进度

【Q5: 对处理结果不满意怎么办？】
A: 
1. 可以提交申诉
2. 提供更详细的说明和证据
3. 升级由高级客服处理
4. 如仍有异议，可联系消费者协会

八、账户安全相关
================================================================================

【Q1: 如何设置支付密码？】
A: 
1. 进入"账户设置"
2. 点击"支付密码"
3. 输入6位数字密码
4. 再次确认密码
5. 设置成功

【Q2: 如何绑定手机号？】
A: 
1. 进入"账户设置"
2. 点击"手机绑定"
3. 输入手机号码
4. 输入验证码
5. 绑定成功

【Q3: 账户被盗怎么办？】
A: 
1. 立即联系客服冻结账户
2. 提供身份验证信息
3. 修改绑定手机和邮箱
4. 取消所有未处理的订单
5. 等待账户解冻后修改密码

【Q4: 如何预防账户风险？】
A: 
- 设置强密码（字母+数字+符号）
- 定期更换密码
- 不在公共网络登录
- 不向他人透露验证码
- 开启账户安全验证
