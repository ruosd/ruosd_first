# 管理员前端界面使用说明

## 🎯 功能概述

管理员前端界面为企业级Agent多记忆系统提供了完整的管理功能：

### 1. 管理员认证
- 安全的登录系统，只有管理员可以访问
- Token认证机制，保护管理功能
- 自动登录状态管理

### 2. 知识库上传
- 支持拖拽上传文件
- 支持 PDF、TXT、DOCX 格式
- 自动文本切割和向量化
- 可选择目标集合

### 3. 数据库查看器
- 查看所有向量集合
- 搜索和浏览文档内容
- 删除指定文档
- 查看集合统计信息

### 4. 记忆管理系统
- 查看各类型记忆统计
- 关键词检索记忆
- 清理过期记忆

### 5. 系统设置
- 查看Ollama配置
- 查看ChromaDB配置
- 查看记忆系统参数
- 系统健康状态监控

## 🚀 快速开始

### 1. 启动前端开发服务器

```bash
cd frontend-vue
npm install
npm run dev
```

### 2. 访问应用

打开浏览器访问：`http://localhost:3000`

### 3. 进入管理界面

点击侧边栏顶部的 "🔐 管理" 按钮，进入管理员登录页面。

### 4. 登录

使用默认管理员账号登录：
- **用户名**: `admin`
- **密码**: `admin123`

> ⚠️ 生产环境请务必修改默认密码！

## 📱 界面布局

### 客服界面（普通用户）
- 左侧：侧边栏（会话信息、操作按钮、系统信息）
- 右侧：聊天主区域

### 管理员界面
- 顶部：导航栏（标题、欢迎信息、退出按钮）
- 中部：统计概览（4个统计卡片）
- 下方：功能标签页和内容区域

## 🔧 功能详细说明

### 知识库上传

1. **准备文件**
   - 支持格式：PDF、TXT、DOCX
   - 文件大小建议不超过10MB

2. **上传步骤**
   - 拖拽文件或点击选择
   - 选择目标集合（默认：knowledge_base）
   - 点击"上传并处理"按钮
   - 等待处理完成

3. **上传结果**
   - 显示文档ID
   - 显示切割的章节数
   - 显示生成的文本块数
   - 显示导入状态

### 数据库查看器

1. **查看集合列表**
   - 点击"刷新列表"按钮
   - 所有集合以卡片形式展示
   - 点击集合卡片查看详情

2. **搜索文档**
   - 输入搜索关键词
   - 点击"搜索"按钮
   - 查看搜索结果（按相似度排序）

3. **删除文档**
   - 在搜索结果或浏览列表中找到目标文档
   - 点击"删除"按钮
   - 确认删除操作

### 记忆管理

1. **查看记忆统计**
   - 显示4种记忆类型的数量
   - 点击记忆类型卡片查看详情

2. **检索记忆**
   - 输入关键词
   - 点击"检索"按钮
   - 查看相关记忆上下文

3. **清理过期记忆**
   - 设置清理天数
   - 点击"清理过期记忆"按钮
   - 查看清理结果

## 🔐 安全性说明

1. **认证保护**
   - 所有管理接口都需要JWT Token认证
   - Token存储在localStorage中
   - 自动路由守卫保护管理页面

2. **权限控制**
   - 普通用户只能访问客服聊天界面
   - 管理员功能需要单独登录
   - 未登录用户无法访问管理界面

3. **密码安全**
   - 默认密码仅为演示使用
   - 生产环境必须修改为强密码
   - 建议定期更换密码

## 📊 API接口

管理接口通过 `/admin` 前缀提供RESTful API：

| 接口 | 方法 | 描述 |
|------|------|------|
| `/admin/login` | POST | 管理员登录 |
| `/admin/collections` | GET | 获取集合列表 |
| `/admin/collections/{name}/stats` | GET | 获取集合统计 |
| `/admin/collections/{name}/search` | GET | 搜索集合文档 |
| `/admin/memory/stats` | GET | 获取记忆统计 |
| `/admin/memory/query` | POST | 检索记忆 |
| `/admin/memory/cleanup` | DELETE | 清理过期记忆 |
| `/admin/documents/process` | POST | 处理文档 |

详细的API文档请访问：`http://localhost:8000/docs`

## 🛠️ 故障排查

### 1. 无法登录
- 检查后端服务是否运行
- 检查管理员账号密码是否正确
- 查看浏览器控制台错误信息

### 2. 上传失败
- 确认文件格式是否支持
- 检查文件大小是否超限
- 确认ChromaDB服务是否正常运行

### 3. 无法查看数据
- 检查Ollama服务是否运行
- 确认向量模型是否已加载
- 查看后端日志排查问题

### 4. 界面显示异常
- 清除浏览器缓存
- 检查是否安装了必要的依赖
- 确认Node.js版本符合要求（>=16）

## 🎨 自定义配置

### 修改默认管理员账号

在后端代码中修改 `admin_router.py` 的认证逻辑：

```python
# 添加新的管理员账号
ADMIN_ACCOUNTS = {
    "admin": "admin123",
    "custom_admin": "custom_password"
}
```

### 修改界面主题

在 `style.css` 中修改CSS变量：

```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  /* ... 其他样式变量 */
}
```

### 添加新功能

1. 在 `src/components/` 创建新组件
2. 在 `admin_router.py` 添加对应API
3. 在 `src/api/admin.js` 添加接口方法
4. 在 `AdminDashboard.vue` 中集成新组件
